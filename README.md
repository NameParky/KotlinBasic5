# 💡 현재 프로젝트의 목표
>1. 동적으로 id값을 불러와서 처리를 할 수 있다.
>2. 에디트 텍스트를 활용하여 입력값을 받아낼 수 있다.
>3. 받아낸 입력값을 연산하고 연산값을 다른페이지에 활용할 수 있다.
>4. 카메라, 갤러리앱을 활용하여 이미지뷰에 이미지를 추가할 수 있다..
>5. 받아낸 이미지를 다른페이지에서 활용할 수 있다.
